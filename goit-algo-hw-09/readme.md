# Висновки

1. **Ефективність жадібного алгоритму**:
   - Жадібний алгоритм виконується досить швидко і просто, оскільки на кожному кроці він вибирає найбільший доступний номінал монети.
   - Проте ефективність жадібного алгоритму залежить від конкретної задачі. У нашому випадку жадібний алгоритм показав досить ефективний результат.

2. **Ефективність алгоритму динамічного програмування**:
   - Алгоритм динамічного програмування більш універсальний і може бути використаний для різних типів задач, включаючи ті, що важко розв'язувати жадібним методом.
   - Хоча алгоритм динамічного програмування може бути складним у реалізації, він зазвичай дає більш оптимальні результати, особливо при роботі з великими сумами або складними умовами.

3. **Порівняння ефективності**:
   - У нашому випадку обидва алгоритми показали задовільні результати. Жадібний алгоритм був досить швидким і простим у виконанні, але алгоритм динамічного програмування показав більш оптимальні результати, забезпечуючи мінімальну кількість монет для досягнення заданої суми.

## **Отже**:
   - При роботі з невеликими сумами або простими умовами обидва алгоритми можуть бути ефективними.
   - Однак у складних задачах або при роботі з великими сумами алгоритм динамічного програмування може бути більш вигідним, оскільки він шукає оптимальні рішення з урахуванням попередніх результатів.
